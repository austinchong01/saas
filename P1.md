# Phase 1 Brand/Creator Platform Outline

## Core Functionalities
- Search Bar
    - by username (handle_name)
- Creator Search Filtering
    - followers (min, max)
    - median_views (min, max)
    - price (min, max)
       - currency (required w/ price)
    - audience demographic (country, gender ratio, age)
    - content type (beauty, sports, gaming, comedy, etc.. )
    - country (US, Korea) *for now
    - languages (English, Spanish, etc.)
- Authentication
